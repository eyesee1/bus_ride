============
Installation
============


Stable release
--------------

To install BusRide, run this command in your terminal:

.. code-block:: console

    $ pip install bus_ride


Or

.. code-block:: console

    $ uv add bus_ride


Or

.. code-block:: console

    $ poetry add bus_ride


These are the preferred methods to install BusRide, as it will always install the most recent stable release.
